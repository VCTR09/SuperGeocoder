# Super Geocoder

####Geography Web App: 


Пользователь загружает CSV-файл в котором есть колонка «Address/address» с адресами мест. Серверная часть приложения - Python скрипт читает файл и добавляет значения широты и долготы к каждому из мест, которые вычисляются из колонки «Address/address».


Таблица результатов отображается на веб-странице, при нажатии пользователем кнопки «Submit». После чего пользователь сможет загрузить CSV-версию файла, содержащего данные широты и долготы. Это можно сделать с помощью кнопки «Download», которая отобразится под таблицей результатов.


В дополнение к этому, при нажатии на кнопку «View on a Map», которая также отобразится под таблицей, будет показана интерактивная карта местности с адресами таблицы показанными на карте в качестве меток. 


Если пользователь загружает какой-либо другой файл (например, без столбца «Address/address»), будет показано сообщение: «Пожалуйста, убедитесь, что у вас есть колонка адресов в файле CSV!».
